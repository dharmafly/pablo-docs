---
category: attr
heading: attr(attributes)
---

Sets multiple attributes (specified as an object), for all elements in the collection and returns the collection.

    var paper = Pablo(demoElement).svg({height:220}),
    rect  = paper.rect();

    rect.attr({
        x: 50,
        y: -50,
        width: 200,
        height:100,
        fill: 'orange',
        transform: 'rotate(45)'
    });
