heading: Pablo(selectors)
category: pablo
---

Creates a Pablo collection, wrapping elements specified by a CSS selector or comma-separated list of selectors.

    // A single element
    Pablo('#foo');

    // Multiple elements
    Pablo('.bar');

    // Multiple selectors
    Pablo('circle, line, #foo, .bar');

    // Complex selectors
    Pablo('#paper g.foo > circle[data-foo=bar]:first-child')

Both SVG and HTML elements can be targeted. Pablo uses the browserâ€™s native selector engine, and so accepts any selector that the browser supports. It uses [Element.querySelectorAll](https://developer.mozilla.org/en-US/docs/DOM/Element.querySelectorAll) under the hood.
