--- 
category: markup
heading: markup()
---

Returns a string of markup for all the elements the collection. Like innerHTML for both HTML and SVG.

    var shapes = Pablo([
        Pablo.circle({r:50}),
        Pablo.rect({x:10})
    ]);

    alert(shapes.markup());


See also [`Pablo(markup)`](/api/pablo/#pablo-06), which creates a collection of elements from SVG markup.
