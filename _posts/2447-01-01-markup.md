---
category: markup
heading: markup()
path: api/markup
---

Returns a string of SVG or HTML markup for all the elements the collection.

    var shapes = Pablo([
        Pablo.circle({r:50}),
        Pablo.rect({x:10})
    ]);

    alert(shapes.markup());


See also [`Pablo(markup)`](/api/pablo/#Pablo-markup-attributes), which creates a collection of elements from SVG markup.
